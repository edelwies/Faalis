Overview

This Concern adds support of dependencies to scaffold using deps key in json file. You can provide a list of scaffold dependencies which should mention as dependecy of Angularjs module of scaffold. Example:

  ....
  "deps": [
      "module1",
      "module2"
  ],
  ...
